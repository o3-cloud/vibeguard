This file lists the current maintainers of the VibeGuard project. Maintainers are responsible for reviewing contributions, making decisions about project direction, and ensuring the project's health and sustainability.

## Becoming a Maintainer

If you're interested in becoming a maintainer, the best path is to:

1. Become an active contributor
2. Review pull requests and help triage issues
3. Demonstrate understanding of the codebase and project goals
4. Show sustained commitment over time

See [GOVERNANCE.md](GOVERNANCE.md) for the full process.
